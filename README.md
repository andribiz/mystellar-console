# mystellar-id Backend Console 

https://mystellar.id backend console

# Technology stack
1. NextJS
2. Firebase hosting
3. Firebase function
4. Firebase Auth
5. lots of nodejs library

# Development

######Installation
Please install nodejs, yarn and firebase tools cli.
```
yarn install
```
######Running

```
yarn next-dev
```
NextJS server will start in `localhost:3000`
